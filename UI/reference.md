\## 01

\### 홈 화면

\- 헤더: 사용자명을 넣은 인삿말, 알림버튼: 알림 목록 페이지로 이동

\- 오늘의 루틴 카드 슬라이더: 하나의 루틴 당 카드 1개. 가로로 슬라이드. 일괄 완료 클릭 시 모든 개체 해당 루틴 완료로 저장. 개별 완료 클릭 시 별도 바텀시트 오픈(06e)

\- 내 개체: 개체 리스트 업데이트 순으로 가로로 나열. 가로 스크롤

\- 최근 기록: 업데이트 순 기록 리스트( 5개 정도) 섹션 상단에 더보기 버튼



\## 02

\### 내 개체 관리 - 개체

\- 상단에 메뉴 이름 및 개체 추가 버튼

\- 개체/루틴 세그먼트 탭

\- 개체 이름 및 종 검색창: onchange로 검색

\- 종 필터칩(많으면 가로스크롤)

\- 개체 카드(2열로 나열) : 개체 사진(없으면 아이콘), 이름, 종, 모프



\## 02b

\### 내 개체 관리 - 루틴

\- 상단에 루틴 추가 버튼

\- 개체/루틴 세그먼트 탭

\- 루틴 타입 필터칩 (전체, 피딩, 청소, 체중, 사용자)

\- 루틴 리스트: 드롭다운 형식 영역 확장

\-- 루틴 헤더: 루틴아이콘, 제목, 간격, 시간, 다음 알림시간, on/off 토글바

\-- 루틴 바디: 1) 루틴 설정 2) 오늘의 상태(진행도 프로그래스바, 개체별 체크 처리 - 클릭시 바텀시트로 개체 리스트(02c) 오픈. 3) 캘린더: 기록이 있는 날짜만 진하게 표시. 날짜 클릭하면 해당 일 완료개체/미완료개체 확인 가능



\## 06e

\### 피딩 기록 바텀시트

* 상단에 제목
* 기록중인 개체
* 급여 종류 (사용자 입력, 또는 기존 사용자가 입력했던 텍스트 중 선택)
* 급여량
* 메모
* 완료 버튼: 완료 버튼을 눌러야만 위에 입력한 내용들 저장됨. 아무 내용도 입력하지 않아도 그냥 완료 상태 저장됨. 완료 누르면 미완료 버튼으로 바뀜. 미완료 버튼 누르면 원래 내용 전체삭제.
* 다음 개체로 넘어가기 버튼
* 홈 화면 루틴 카드에서 넘어온 경우 뒤로 버튼과 닫기 버튼은 생략. FAB 버튼을 통해 넘어온 경우 뒤로 버튼(개체선택으로 되돌아감.  완료 및 저장된 개체를 선택 해제할 경우 해당 내용 초기화)
* 닫기 버튼
